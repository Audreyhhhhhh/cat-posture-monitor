# 🐱 猫咪坐姿守护助手
AI 姿态识别坐姿监测工具，实时识别驼背、头前伸、二郎腿，支持语音提醒、飞书推送、数据统计与历史记录。

## ✨ 功能
- AI 姿态关键点识别
- 实时监测：驼背 / 头前伸 / 二郎腿
- 定时检测（默认5分钟）
- 语音提醒
- 飞书机器人推送
- 历史数据统计
- 猫咪主题高颜值网页看板
- 日期切换查看记录

## 🚀 技术栈
- Python
- OpenCV
- MediaPipe（AI姿态模型）
- HTTP 网页服务

## 📸 效果
- 实时摄像头姿态识别
- 不良坐姿自动提醒
- 每日数据统计
- 网页端可视化看板

## 🎯 项目说明
本项目基于深度学习姿态估计模型，属于标准计算机视觉AI应用，可用于学习、办公健康监督。
